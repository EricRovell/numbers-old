# Число

Число, как элемент взаимодействия пользователя, схоже с [символами](/tutorial/ru/sandbox/symbols) из прошлой главе. По сути, __число__ - символы, склееные между собой воедино. В этом и основное отличие между __символами__ и __числом__, оба представляют собой число, но первые разбиты на части, что даёт совершенно иной пользовательский опыт взаимодействия. __Число__ в этом плане более примитивно и прямолинейно.

## Анатомия

Число, как уже было сказанно выше, очень схоже с __символами__. Анатомия __числа__, соответственно, тоже. Оно состоит из двух основных блоков: элемента ввода и панели управления. 

Элемент ввода прост и проямолинеен; он позволяет вводить и редактировать число напрямую.

> Пользовательский ввод допускает только разрешенные для данной системы счисления символы. В противном случае ввод даёт знать об ошибке ввода.

Панель управления своим функционалом повторяет панель управления __символов__, с единственным отличием: нет необходимости контроля количества символов. Поэтому, в данном случае панель позволяет лишь контроллировать систему счисления для числа в элементе ввода.
