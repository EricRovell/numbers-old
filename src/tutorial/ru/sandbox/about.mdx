# Режим песочницы

Режим песочницы является основным режимом данного приложения. Песочница - это возможность переводить числа в произвольные системы счисления интерактивно. Другими словами - это переводчик чисел из различных систем счисления в красивой обёртке.

При загрузке приложения можно увидеть два основных блока режиме:

1. Множество символов (цифр),
2. Число.

[_Множество символов_](/tutorial/ru/sandbox/symbols) представляет собой упорядоченное множество символов с возможностью изменения состояния с учётом используемой системы счисления. Другими словами, область символов это разбитое на отдельные кирпичики число.

[_Область числа_](/tutorial/ru/sandbox/number) представляет собой непосредственно "цельное" число.

_Множество символов_ и _Число_ связанны между собой в обе стороны: изменение состояния первого влияет на состояние второго. Например, изменив один из символов, число будет преобразовано в соответсвие с состоянием символьного числа.
