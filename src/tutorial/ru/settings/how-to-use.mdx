# Как использовать собственные предпочтения

Пользователь имеет возможность сохранять предпочтения, как и возвращать всё к настройкам по умолчанию. Для этого у каждой формы в самом начале имеются элементы контроля.

Предпочтения сохраняются непосредственно в браузере, используя технологию локального хранилища.
