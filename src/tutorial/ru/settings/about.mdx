# О предпочтениях

Приложением поддерживается возможность редактировать параметры по умолчанию. Доступные параметры делятся на три основных раздела:

- Основные;
- Режим Песочницы;
- Режим Игры (на данный момент в разработке).

## Основные параметры

Основные параметры позволяют сохранить пользователю:

- Язык приложения;
- Тема оформления.

Язык оформления доступен на русском и английском языках. По умолчанию устанавливается английский.
Тема оформления доступна в тёмном и светлом вариациях. По умолчанию устанавливается тёмная.

## Режим Песочницы

Режим песочницы позволяет настроить целый ряд параметров по умолчанию, доступный после запуска или обновления страницы приложения.

### Минимальное и максимальное количество символов

Контролируют границы количества допустимых символов.
Имеются внутренние ограничения в пределах от 0 до 16 включительно.

### Символы по умолчанию

Контролирует количество символов и их значение.
Принимаемые значения зависят от минимального и максимального количества символов, а так же от системы счисления на вход (символы).

### Система счисления

Контролирует степень систем счисления на вход и выход.
По умолчанию принято считать символами входом и число - выходом перевода.

### Репрезентация символов

Контролирует тип и внешний вид символов.
Внешний вид привязан к типу используемого символа.
Некоторые типы символов имеют ограничения пользования, например, лампочки можно использовать только в двоичной системе счисления.

### Подписи

Контролирует тип подписей для верхнего и нижнего блока по умолчанию.

## Режим Игры

> На данный момент режим находится на стадии разработки.
Информация будет своевременно обновлена.
