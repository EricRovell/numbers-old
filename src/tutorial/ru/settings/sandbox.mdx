# Настройки. Режим песочницы

Режим песочницы позволяет контролировать состояние символов, значением системы счисления, репрезентацией и подписями.

## Символы

Данный раздел контролирует количество используемых символов и их значения по индивидуальности.

> Количество символов ограничено внутренними параметрами приложения. Текущее значение позволяет использовать значения в промежутке от __0__ до __16__. Ввод значений, выходящих за данные пределы будет сводиться к значениям по умолчанию автоматически.

Значение символов по умолчанию можен быть задано вручную. При вводе следует руководствоваться следующим:

- Количество символов должно находиться в заданных пределах, установленных пользователем, либо ограничениями приложения;
- доступные значения символов зависят от используемой системы счисления;
- разделителем при вводе могут быть пробел, тире и запятая, допустимо смешение разделителей в одном вводе.

> Ввод проходит проверку на соблюдение всех вышеописанных условий. При несоответсвии данных ошибки будут автоматически приведены к ближайшему адекватному значению параметров. Важно помнить, что проверка на соотвествие проходит после проверки количества символов и значения системы счисления.

При несоответсвии количеству символов часть символов будет либо отброшена, либо приписана (нули по умолчанию). Недопустимые символы - приведены к ближайшему допустимому значению.

## Система счисления

Данный раздел контролирует значения систем счисления по умолчанию для __Символов__ и __Числа__. 

> Допустимые значения ограничены внутренними параметрами приложения. Допустим отрезок от __2__ до __36__. 

Значения системы счисления находятся в зависимости от режиме/репрезентации __Символов__. При использовании "Лампочек", например, никакая система, кроме двоичной не имеет смысла. При несоответствии ввода режиму __символов__ значение системы счисления будет переведено в двоичную автоматически. 

## Режим / Репрезентация символов

Данный раздел позволяет выбрать используемый по умолчанию режим отображения символов. 

При выбора режима "Лампочки" необходимо иметь ввиду, что данный режим имеет смысл лишь при двоичной системе счисления.
