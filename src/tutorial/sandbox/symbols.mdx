# Режим песочницы. Символы

Символы - упорядоченное множество символов, используемых для записи итогового числа. Простыми словами, это число, разбитое на "атомы". Может возникнуть резонный вопрос: "А почему, собственно, символы? Почему не числа?". Есть две причины. Рассмотрим их на примере систем счисления больше десяти:

1. Число 11 в системе счисления от 11 и больше требует для записи либо свой собственный символ, либо выделения самой записи "11" как отдельное обозначение, нежели ли число. 

2. Исходя из выженаписанного, при работе с системами счисления больше десяти по обыкновению используются буквы латинского алфавита, например, 11 станет "A". Поэтому, для избегания путанницы, что число состоит из цифр и букв, в приложении число состоит из символов и раздел по работе с разбитым числом называется _"Символами"_.

Рассмотрим работу с символами непосредственно на практике.

По умолчанию символы представленны в виде блоков, состоящих из следующих элементов:

- Верхняя вспомогательная подпись,
- Кнопка увеличения символа "+1",
- Значение символа,
- Кнопка уменьшения сисвола "-1",
- Нижняя вспомогательная подпись.

Значение символа может быть изменено двумя способами: вводом значения напрямую, либо с помощью элементов управления увеличения/уменьшения значения, расположенные непосредственно над и под символом.

> Исключением является режим двоичных чисел, пользовательский ввод заменён на переключение при интеракции с пользователем.

> Пользовательский ввод проходит проверку на "адекватность", поэтому нелегитимные значения не примутся и символ даст об этом знать. Стоит попробовать.
