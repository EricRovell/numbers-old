# Непозиционные системы счисления

При возникновении необходимости счёта - возникла потребность записи чисел. 

Изначально, всё было просто: зарубка на какой-либо поверхности соответствовала одному предмету, например, одному фрукту. Так появилась первая система счисления - единичная.

## Единичная система счисления

Число в данной системе счисления представляет собой строку из засечек, либо множество элементов, количество которых принято значению этого числа. Таким образом, урожай из сотни бананов будет равен число, состоящих из сотни засечек, либо сотни камешков, к примеру.

Единичная система обладает явными неудобствами:

- чем больше число, тем длиннее строка засечек, тем больше камешков (либо других объектов) необходимо для представления числа;
- при больших числах (от 5 и выше) сложно быстро определить число;
- растёт вероятность ошибки при работе с числами.

В процессе развития системы появились группировки по 3, 5 и 10 штук. При этом каждой группировке соотвествовал определённый сисвол, либо предмет. Так как для подсчёта изначально использовались пальцы рук, первые символы появились для групп из 5 и 10 единиц. Это позволило создать более удобные системы записи чисел.
