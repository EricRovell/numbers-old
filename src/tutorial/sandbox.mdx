# Режим песочницы

Режим песочницы является основным режимом данного приложения. Песочница - это возможность переводить числа в произвольные системы счисления интерактивно. Другими словами - это переводчик чисел из различных систем счисления в красивой обёртке.

При загрузке приложения можно увидеть два основных блока режиме:

1. Область символов (цифр),
2. Область числа.

_Область символов_ представляет собой упорядоченное множество символов с возможностью изменения состояния с учётом используемой системы счисления. Другими словами, область символов это разбитое на отдельные кирпичики число.

_Область числа_ представляет собой непосредственно "цельное" число.
